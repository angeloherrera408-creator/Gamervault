REST API SPECIFICATIONS

Documentación de Endpoints operativos bajo el servidor Spring Boot (Port: 8080).

POST /api/auth/login

// Procesa la llave de seguridad corporativa y genera la sesión activa.

Parámetros del Body (application/json):
Campo Tipo Obligatorio
username String SÍ (admin)
password String SÍ (12345)
Payload Requerido: 
{
  "username": "admin",
  "password": "█████"
}
Respuesta Exitosa (200 OK): 
{
  "status": "authenticated",
  "sessionToken": "gv_token_f89342a1b9",
  "username": "admin"
}
Respuesta Errónea (401 Unauthorized): 
{
  "status": "error",
  "message": "Usuario o clave incorrectos."
}
